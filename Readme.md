## Primera prueba de repositorio a trevez de consola🤖: 
Esta es una de las primeras pruebas al subir un repositorio a git a partir de consola, se utilizo la guia de los archivos compartidos por el discord para realizar dichos pasos. 



## Pasos a seguir🤯:
- Configuracion del usuario a partir de los comandos: Comando : git config --global user.name “SputnikCAMPUS”
Comando : git config --global user.email
“Email-deseado”
Comando : git config --global init.defaultBranch main

- Creacion de repositorio vacio en una ubicacion deseada: GIT INIT

- Se añade todos los datos con: git add.
